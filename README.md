# rmqr_wallpaper
Create meeting-wallpaper using rmqrcode

```bash
python3 generate.py \
    --output ./data.png \
    --text "https://github.com/Ar-Ray-code/rmqr_wallpaper" \
    --wallpaper ~/Downloads/03eighthouse07.jpg \
    --mode "16:9" \
    --flip
```
